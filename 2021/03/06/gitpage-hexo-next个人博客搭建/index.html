GitHub Pages+Hexo+NexT 个人博客搭建

发表于 2021-03-06 分类于 技术分享 ， 博客

这篇文章将分享一个从零开始的自建个人博客网站的过程。

开始前的一些话

为什么要建个人博客？

让自己养成写博客的好习惯。

记录一下自己的学习过程。

为什么选择自建?

市面上虽然已经有了像简书、CSND、博客园等成熟的内容创作平台，但是我还是选择了自建个人博客。主要原因在于那些平台广告繁杂、还有太多我用不到的功能，每次我看到了都感到很厌烦。

我的要求很简单：方便写作、方便查找、看着舒服。

因此我选择自建博客，仅加入我自己需要的功能，一切从简。

为什么选择这套技术方案？

GitHub Pages技术平台，Hexo博客框架，NexT主题。

为什么是GitHub Pages？

免费、轻量级、支持Markdown

GitHub Pages可以免费使用，可以直接在上面部署，不需要自己去租用一台服务器。

我本人习惯在本地用Markdown写一些记录，因此GitHub Pages恰好符合我的需求。

为什么是Hexo？

正如Hexo官网所介绍的，Hexo是一个快速、简洁且高效的博客框架，高度符合我的需求。

为什么是NexT？

主题这种东西就是仁者见仁智者见智的了。NexT精致简约，符合我的审美。
