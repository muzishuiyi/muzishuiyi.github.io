1047.删除字符串中的素有相邻重复项

给出由小写字母组成的字符串 S，重复项删除操作会选择两个相邻且相同的字母，并删除它们。

在 S 上反复执行重复项删除操作，直到无法继续删除。

在完成所有重复项删除操作后返回最终的字符串。答案保证唯一。

示例：

提示：

  1. 1 <= S.length <= 20000
  2. S 仅由小写英文字母组成。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/remove-all-adjacent-duplicates-in-string

题解

栈

本题可以用栈的特点来进行解题。

从字符串 S 中弹出第一个字符 c，如果栈为空，则将字符 c 入栈。若栈不为空，则比较栈顶字符和字符 c ，若相同，则将栈顶字符弹出，否则将字符 c 入栈。

重复上述操作直至字符串 S 为空。

若字符串 S 的长度为 n，则时间复杂度为$O(n)$。

快速解法

通过字符数组和指针移动实现。

知识点

Java StringBuffer 和 StringBuilder 类
